# Tic Tac Toe

A simple browser-based Tic Tac Toe game built with HTML, CSS and JavaScript.

## Demo

Open `index.html` in a browser to play locally. For a local development server, run `npx http-server` or serve the project folder with any static server.

## Features

- Two-player local game (Player X and Player O)
- Visual highlights for winning combination
- Reset / New game button
- Lightweight: no build step or external dependencies

## Project structure

- `index.html` — game UI
- `styles.css` — styling
- `script.js` — game logic

## How to play

1. Open `index.html` in your browser.
2. Players take turns clicking empty cells to place their mark (X or O).
3. The first player to get three marks in a row (horizontal, vertical or diagonal) wins.
4. Click the Reset/New Game button to restart.
